# User Acceptance Testing (UAT) Table

This document organises user acceptance testing by system feature rather than by individual specification item. Each row describes one feature of the implemented system and lists the specification requirements that the feature satisfies. The `Test Result` column is left blank so it can be updated during actual UAT.
